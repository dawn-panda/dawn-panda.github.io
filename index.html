PANDAWN Network

A remotely reconfigurable Rad/Nuc sensing network designed to evolve over time

PANDAWN is an urban sensing network composed of approximately 14 strategically placed sensor nodes across Chicago. The network combines radiation/nuclear sensing, edge computing, cloud-connected analysis, and remote reconfiguration to support long-term multimodal data collection and algorithm development in realistic urban environments.

Distributed urban sensing

  • Strategically placed nodes provide geographically diverse measurements across Chicago.
  • The network supports data collection across varied urban environments and deployment contexts.

Built-in edge computing

  • AI/ML and conventional computing can be performed close to the data source.
  • Edge-local processing enables responsive analysis and reduced dependence on centralized processing.

Software and hardware extensibility

  • The network can be updated over time to support new sensors, workflows, and algorithms.
  • Extensibility is supported within the practical limits of deployed hardware and software.

Closed-loop edge-cloud operation

  • Jobs can be scheduled and coordinated across edge nodes and cloud resources.
  • The architecture supports feedback between cloud-level analysis and edge-level execution.

Context-aware execution

  • Edge nodes can support adaptive job execution based on local data, system state, or broader network context.
  • This enables more intelligent use of sensing, compute, and communication resources.

Network-wide aggregation and analysis

  • Data from multiple nodes can be aggregated for cloud-based analysis, visualization, and algorithm evaluation.
  • Network-wide views support spatial, temporal, and multimodal analysis across deployment sites.

Project Goals

Advancing multimodal sensing, data curation, and AI-enabled domain awareness

The project builds on the PANDAWN network to collect, curate, and analyze real-world multimodal sensing data across geographically distributed sites. These efforts support data-driven algorithm development, automated labeling, and AI-enabled analysis for nonproliferation-relevant applications.

Acquire, label, and curate PANDAWN multi-sensor data

  • Operate the PANDAWN network for sustained multimodal data collection.
  • Disseminate curated datasets to the NA-22 community.
  • Apply machine learning methods for automatic data labeling.
  • Integrate data science methods into acquisition and curation workflows.

Explore multi-sensor data and radiological pattern-of-life

  • Analyze radiological pattern-of-life across time, location, and environmental conditions.
  • Develop new algorithms for multi-sensor data interpretation.
  • Incorporate validated analysis methods into the labeling and curation pipeline.

Evaluate AI methods for multimodal analysis and domain awareness

  • Assess AI techniques for fusing heterogeneous sensing modalities.
  • Evaluate model performance using realistic, geographically distributed data.
  • Advance domain-awareness capabilities for complex real-world environments.
